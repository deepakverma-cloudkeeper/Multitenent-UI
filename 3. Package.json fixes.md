
## Remove Old or Un-treeshakable packages.
---
| Category | Remove | Keep |
|---|---|---|
| Dates | `moment`, `moment-timezone`, `react-moment`, `bootstrap-daterangepicker`, `mui-daterange-picker` | `dayjs` + `@mui/x-date-pickers` |
| Tables | `react-table` (v7) | `@tanstack/react-table` (v8) |
| Virtualisation | `react-virtual` (v2) | `@tanstack/react-virtual` (v3) |
| Drag & Drop | `react-dnd` + all backends | `@dnd-kit/core` + `@dnd-kit/sortable` |
| UI Framework | `@material-ui/core` (v4) | `@mui/material` (v5) |
| Charts | `fusioncharts` (audit — ~800KB) | Evaluate `recharts` (~130KB) |


### Miscellaneous 
| Package                      | Verdict                                                                                  | Size (min) | Action          |
| ---------------------------- | ---------------------------------------------------------------------------------------- | ---------- | --------------- |
| lodash.clonedeep             | Use `structuredClone()` -- built into all modern browsers, zero KB.                      | ~5KB       | Remove          |
| lodash.isequal               | Use `fast-deep-equal` (~1KB) or `JSON.stringify` comparison for simple cases.            | ~5KB       | Remove          |
| react-loadable               | Unmaintained since 2018. React 18 has React.lazy() + Suspense built in natively.         | ~3KB       | Remove          |
| immutability-helper          | Immer is the modern standard and more readable. Or just use spread operators.            | ~3KB       | Evaluate        |
| choices.js                   | Vanilla JS select enhancement — not React-native. Likely a legacy leftover. Audit usage. | ~20KB      | Audit           |
| redux-mock-store             | Test-only dep — should be in devDependencies, not dependencies. Move it.                 | ~2KB       | Move to devDeps |
| @testing-library/react-hooks | Merged into @testing-library/react v13+. We already have that. Remove this.              | ~3KB       | Remove          |


|Library|Size|Speed|
|---|---|---|
|fast-deep-equal|Tiny|Very fast|
|lodash.isEqual|Larger|Slower|
|JSON.stringify compare|Unsafe|Sometimes fast|
